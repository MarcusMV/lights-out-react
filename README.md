## LGIHTS OUT - Unplug Psoriasis 

#### Project:

Frontend react application, built with React, Redux, JavaScript, and CSS.

## Installation and Setup Instructions

You will need node and npm installed globally on your machine.

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
